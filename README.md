# GreedyRaAlgorithm

Greedy randomized adaptive algorithm to solve travelling salesman.

## Installation

Add this line to your application's Gemfile:

    gem 'greedy_ra_algorithm'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install greedy_ra_algorithm
